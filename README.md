# blackjack-buddy

A blackjack application where users can play against the computer. The application should have a simple and intuitive interface, allowing users to easily understand how to play. It should feature a deck of cards, the ability for users to hit or stand, and automatic scoring against the computer's hand.

## Setup

```sh

cd blackjack-buddy
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
